# TrustWork Dominicana Portfolio Landing

Landing page empresarial inspirada en el lenguaje visual de sitios premium tipo Prediktive, adaptada al posicionamiento institucional de TrustWork Dominicana.

## Incluye

- Hero principal con propuesta de valor clara
- Bloques de confianza y diferenciación
- Portafolio de servicios
- Sección sobre República Dominicana como hub operativo
- Equipo operativo y enfoque B2B
- Sectores atendidos
- CTA final listo para despliegue en Vercel

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Stack

- React
- Vite
- lucide-react
